TransactionTypes API

Criar tipo de transação

POST /transaction_types

Parameters

Name Description
transaction_type[code] Transaction type code
transaction_type[description] Transaction type description
transaction_type[income] Transaction type income

Request

Route

POST /transaction_types

Body

transaction_type[code]=1&transaction_type[description]=Descri%C3%A7%C3%A3o+da+Opera%C3%A7%C3%A3o&transaction_type[income]=true

cURL

curl "localhost:3000/transaction_types" -d 'transaction_type[code]=1&transaction_type[description]=Descri%C3%A7%C3%A3o+da+Opera%C3%A7%C3%A3o&transaction_type[income]=true' -X POST \
	-H "Content-Type: application/x-www-form-urlencoded"

Response

Status

201 Created

Body

{
  "id": 26,
  "code": 1,
  "description": "Descrição da Operação",
  "income": true,
  "created_at": "2020-08-30T14:54:06.824Z",
  "updated_at": "2020-08-30T14:54:06.824Z"
}
