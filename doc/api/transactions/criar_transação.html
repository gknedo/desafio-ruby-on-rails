Transactions API

Criar transação

POST /transactions

Parameters

Name Description
transaction[store_id] Transaction store
transaction[transaction_type_id] Transaction transaction type
transaction[value] Transaction value
transaction[cpf] Transaction cpf
transaction[card] Transaction card
transaction[transacted_at] Transaction transacted at

Request

Route

POST /transactions

Body

transaction[store_id]=140&transaction[transaction_type_id]=144&transaction[value]=99999&transaction[cpf]=99999999999&transaction[card]=9999999999999999&transaction[transacted_at]=2020-09-10+01%3A57%3A27+UTC

cURL

curl "localhost:3000/transactions" -d 'transaction[store_id]=140&transaction[transaction_type_id]=144&transaction[value]=99999&transaction[cpf]=99999999999&transaction[card]=9999999999999999&transaction[transacted_at]=2020-09-10+01%3A57%3A27+UTC' -X POST \
	-H "Content-Type: application/x-www-form-urlencoded"
